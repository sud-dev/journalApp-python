Personal Journal  Application using Python

Steps to execute the code :

----> Extract Sudhanshu_journalApp.zip folder to any path
----> Go to location where Sudhanshu_journalApp is located (path where zip file is extracted in previous step)
----> type cmd in address bar 
----> command prompt window will be opened in the current location of Sudhanshu_journalApp folder
----> run command : py journalApp.py
